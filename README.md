# Buzzing Intelligence: Acoustic Classification of Agricultural Insects

A deep learning project that classifies agricultural insect species from audio recordings using log-mel spectrograms and a convolutional neural network (CNN).

---

## Overview

This project trains a CNN to identify insect species from their acoustic signatures. Raw WAV audio files are converted into log-mel spectrograms, stored as TFRecords, and fed into a 4-block CNN classifier.

**Dataset:** [InsectSound1000](https://www.kaggle.com/datasets/hesi0ne/insectsound1000) — 1,000 insect sound recordings across 12 species, sourced from Kaggle.

---

## Pipeline

```
WAV audio files
     ↓
Log-Mel Spectrograms (.npy)
     ↓
Stratified Group-Aware Train/Val/Test Split (≈70/15/15%)
     ↓
TFRecords
     ↓
CNN Training (with SpecAugment)
     ↓
Evaluation (Accuracy, Top-3 Accuracy, Confusion Matrix)
```

---

## Model Architecture

A 4-block CNN followed by a dense classifier:

| Layer | Details |
|---|---|
| Conv Block 1 | Conv2D(32, 8×8) → BatchNorm → ReLU → MaxPool |
| Conv Block 2 | Conv2D(64, 8×8) → BatchNorm → ReLU → MaxPool |
| Conv Block 3 | Conv2D(128, 8×8) → BatchNorm → ReLU → MaxPool |
| Conv Block 4 | Conv2D(128, 8×8) → BatchNorm → ReLU → GlobalAvgPool |
| Classifier | Dropout(0.3) → Dense(128, ReLU) → Dropout(0.3) → Dense(12, Softmax) |

**Key hyperparameters:**
- Input shape: 128 × 256 × 1 (H × W × channels)
- Batch size: 32
- Optimizer: AdamW (lr=1e-3, weight_decay=1e-4)
- Max epochs: 30 (with early stopping, patience=6)

---

## Data Augmentation

[SpecAugment](https://arxiv.org/abs/1904.08779) is applied during training:
- Random time masking (width 10–30 frames, applied with 50% probability)
- Random frequency masking (width 5–15 bins, applied with 50% probability)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/BuzzingIntelligence.git
cd BuzzingIntelligence
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Set up Kaggle credentials

To download the dataset automatically via `kagglehub`, you need a Kaggle API key.

1. Go to [https://www.kaggle.com/settings](https://www.kaggle.com/settings) → API → Create New Token
2. Place the downloaded `kaggle.json` in `~/.kaggle/kaggle.json`
3. Run `chmod 600 ~/.kaggle/kaggle.json` (Linux/Mac)

### 4. Run the notebook

Open `BuzzingIntelligence_1.ipynb` in Jupyter and run all cells in order:

```bash
jupyter notebook BuzzingIntelligence_1.ipynb
```

The notebook will:
1. Download the dataset from Kaggle
2. Generate log-mel spectrograms → `LogMelSpectrograms/`
3. Split data and write TFRecords → `TFRecords/`
4. Train the CNN and save the best checkpoint → `best_model.keras`
5. Evaluate on the test set and display the confusion matrix
